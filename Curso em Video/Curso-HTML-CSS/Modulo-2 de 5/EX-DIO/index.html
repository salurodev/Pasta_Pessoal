Tags Semânticas

O que é Tag Semântica?

O HTML semântico é a forma de deixar o site com suas informações bem explicadas e compreensíveis para o computador, ajudando até mesmo em sua busca no Google e facilitando o entendimento de leitores de acessibilidade. Com o HTML semântico, ficou muito mais fácil de interpretar páginas.

Elementos Semânticos

Os elementos semânticos são aqueles usados para definir diferentes partes de uma página da web, ou seja, eles tem um significado. Uma tag <div> , por exemplo, não nos diz nada sobre qual será o conteúdo do elemento. Já a tag <footer> , nos diz exatamente qual será o conteúdo deste elemento.
